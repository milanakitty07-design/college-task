## Django проект 2 - Minisite

### Описание проекта:

- Страница регистрации (signup/)

### Запуск проектов:
```bash
# Запуск minisite
cd minisite
python manage.py runserver

# Запуск DiasDjangoProject
cd DiasDjangoProject
python manage.py runserver
```

После запуска серверов сайты будут доступны по адресу: http://127.0.0.1:8000/
